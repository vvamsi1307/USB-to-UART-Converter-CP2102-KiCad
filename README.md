# USB to UART Converter using CP2102

## Project Overview
This project is a compact USB to UART converter designed using KiCad.  
The board allows communication between a computer USB port and microcontrollers via UART.

## Features
• CP2102 USB to UART Bridge Controller  
• USB Micro Connector Interface  
• UART TX/RX Header Output  
• ESD Protection Diodes  
• Decoupling Capacitors for Stable Operation  
• 2-Layer PCB with Ground Plane  
• Compact PCB Layout  

## Applications
• Embedded System Debugging  
• Microcontroller Programming  
• Serial Communication Interface  
• IoT Development Boards  

## Tools Used
